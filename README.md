# gallery.html Project

## Description
This is a website that lets users slide images forward and backward. It works on both computers and phones.

## Features
- Slide images forward and backward
- Works on desktop and mobile
- Easy buttons to navigate
- Smooth transitions between images

## How to Run
1. Download or clone the project
2. Open `index.html` in a web browser
3. Click the "Next" and "Prior" buttons to see images

## Technologies Used
- HTML
- CSS
- JavaScript
